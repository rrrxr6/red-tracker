Description
Easily identify posts with Rioter comments and then quickly navigate through a post's comment section to find Rioter comments with the /r/LeagueOfLegends Red Tracker Chrome extension. All posts with Rioter comments are turned red. Once in the comments section, an unobtrusive navigation menu lets you scroll directly to Rioter comments. Finally, /r/LeagueOfLegends Red Tracker is compatible with Reddit Enhancement Suite.
